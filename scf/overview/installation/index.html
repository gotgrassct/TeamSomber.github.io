Step 1: Go to https://fab.com to purchase and download my plugin (if you haven't already) Step 2: Install it via the Epic Games Launcher. Step 3: That's it! You're ready to start implementing it.

Please go to the Quick Start Guide to quickly get started on fully setting all the systems up from scratch on a brand new character (can also follow the tutorial on an existing character)
